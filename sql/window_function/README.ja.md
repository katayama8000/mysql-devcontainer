# ウィンドウ関数

このディレクトリには、SQLウィンドウ関数の例が含まれています。

## `window_function_example.sql`

このファイルは、以下を含むいくつかのウィンドウ関数の使用法を示しています。

- `ROW_NUMBER()`: パーティション内の行の順序に基づいて、パーティション内の各行に一意の連続した整数を割り当てます。
- `RANK()`: パーティション内の各行にランクを割り当てます。ランキング基準で同じ値を持つ行は同じランクを受け取ります。タイの後の次のランクは連続しません。
- `SUM()` をウィンドウ関数として使用: パーティション内の列の累積合計を計算します。

### セットアップ

このスクリプトはまず `sales` テーブルを作成し、いくつかのサンプルデータを挿入します。このテーブルは、ウィンドウ関数をデモンストレーションするために使用されます。

### クリーンアップ

スクリプトには、作成されたテーブルをクリーンアップするために、最後に `DROP TABLE IF EXISTS sales;` ステートメントが含まれています。
